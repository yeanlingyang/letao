bootstrap pc端, 响应式ui框架  (比较重, 大小比较大, 相对来说, 移动端加载比较慢的)

将来实现移动端, 也有专门的 ui框架(ui组件库)
  1. mui 
  2. 妹子ui  amazeUI 
  3. sui 手淘
    sui 基于bootstrap的 pc ui框架
    sui mobile 专门的移动端的 ui 框架

ui框架的操作流程: 1. 引包  2. 复制粘贴结构  3. 自定义样式, 覆盖原有样式, 添加自己的样式


库文件的说明:
1. fa => font-awesome 字体图标库, 引入 css 文件, 就可以使用它提供的字体图标了  (第三方字体图标库)

2. mui 就是一个 ui 框架(ui组件库)

3. zepto 移动端的jquery, 功能和用法和jquery一模一样, 大小小了很多
   实质上是 在 jquery 的功能基础上, 去掉很多平时移动端用不到的功能, 大小体积就小了
